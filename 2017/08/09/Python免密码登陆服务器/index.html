一、需求分析

  • 登录堡垒机
  • 登录服务器
  • 输入命令

二、方案选择

首先我们考虑的是Linux下是可以用expect进行ssh交互的，强大如Python当然也有这个能力，于是我们就自然想到了Python的pexpect库。
如果对pexpect库不熟悉的可以看下面的文章进行学习：pexpect学习教程
